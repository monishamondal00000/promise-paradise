# Image Assets

Place your static wedding images in the appropriate folders:

- **landing/** — Hero carousel images for the landing page (recommended: 1920x1080px)
- **gallery/** — Wedding gallery photos (recommended: 600x800px or 800x600px)
- **packages/** — Package thumbnail/banner images (recommended: 800x600px)
- **destinations/** — Destination cover images (recommended: 800x600px)

## Usage

Reference images with `/images/<folder>/<filename>`, e.g.:
```
/images/landing/hero1.jpg
/images/gallery/ceremony1.jpg
```
